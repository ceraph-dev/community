# Ceraph

Found a bug? [Open an issue](https://github.com/ceraph-dev/community/issues). 

Have a feature request or feedback? [Start a discussion](https://github.com/ceraph-dev/community/discussions).

## Getting Started

Head over to [our website](https://ceraph.dev) to get started, or drop into your React Native project and run:

`npx @ceraph/react-native-mcp@latest init`

## Features

Ceraph installs as an MCP server, so any AI coding agent can drive your app — tap, swipe, screenshot, and surface runtime errors right in your chat. [See what's supported](https://ceraph.dev/#integrations): React Native, Expo, and every major MCP client.
